# Inventory & Order Management System

## Overview
A full-stack inventory and order management platform built using Oracle Database, FastAPI, and JavaScript.

## Features
- User Authentication
- Inventory Management
- Order Processing
- Sales Reporting
- Oracle PL/SQL Procedures
- Triggers and Views
- REST APIs using FastAPI

## Tech Stack
- Oracle SQL
- PL/SQL
- Python (FastAPI)
- HTML
- CSS
- JavaScript

## Database Concepts Used
- Normalization (3NF)
- Triggers
- Views
- Procedures
- Packages
- Cursors
- Transactions
